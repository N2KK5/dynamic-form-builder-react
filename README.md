## Description
A dynamic form builder application where:
- Admin creates forms and fields dynamically
- Users submit form data
- Dashboard displays submitted data
- No backend used (localStorage persistence)

## Tech Stack
- React
- Tailwind CSS
- Vite
- localStorage

## Features
- No hardcoded form fields
- Dynamic rendering based on schema
- Supports dropdown, radio, checkbox
- Admin edits reflect instantly
- Clean UI & reusable components

bun install  
bun run dev
